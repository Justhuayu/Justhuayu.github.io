# Mac 上 iCloud 下载慢

​	Mac 上开启了 icloud 同步云盘内容，但是下载很慢，导致桌面和文稿中文件一直被同步占用，无法进行复制或删除等操作。



# 解决办法

​	设置中关闭 icloud 同步。具体在，设置 >> Apple ID >> iCloud >> iCloud云盘 ，关闭 "同步此 mac" 和 "桌面和文稿文件夹" 选项。
