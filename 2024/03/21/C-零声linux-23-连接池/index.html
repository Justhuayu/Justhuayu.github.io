2. 预处理

​ 每条sql语句请求一次，mysql服务器就要接收并处理一次，当一个脚本文件对同一条语句反复执行多次的时候，mysql服务器压力会变大，所以出现mysql预处理，减轻服务器压力。

  1. prepare 预处理名 from "select * from table where id=？";

  2. set @变量名=值;

  3. execute 预处理名 using @变量名1,@变量名2;

  4. drop prepare 预处理名;
